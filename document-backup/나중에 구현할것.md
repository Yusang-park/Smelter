

### 11-5. Ralph 전환 (Ralph 실행 중에만)

`/ralph` 실행 중인 경우, 현재 Task 완료 후 자동으로 다음 selected task로 전환한다.  
**각 Task는 반드시 독립된 서브에이전트(독립 컨텍스트)로 실행한다.** ([독립 컨텍스트 원칙](#독립-컨텍스트-원칙-context-isolation) 참조)

```
[ralph] ✓ Task {id} complete — moving to next task

features/<slug>/task.md에서 다음 미완료 항목 (- [ ]) 확인
  → 있으면: 필요한 경우 tasker로 더 잘게 쪼갠 뒤 시작
  → 없으면: 전체 큐 완료 보고
```

**Ralph 완료 메시지:**
```
═══════════════════════════════════
 RALPH COMPLETE
 Tasks   : {완료}/{전체}
 Failed  : {실패 수}
 Duration: {총 소요 시간}s
═══════════════════════════════════
```

### 프리셋 적용
- `work`, `default`: 포함
- `tasker`: 건너뜀
